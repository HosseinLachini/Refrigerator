Three Keys Display Panel
------------------------
Three keys display panel designed by ITW APPLIANCE REFRIGERATION PRODUCTS GROUP, with PIC microcontroller. This board used for Larder, 4-Star and Nofrost products in PARS company. I changed MCU to AVR microcontroller and wirte program with C++ to easly change MCU to any microcontroller for example you can use an ARM.

Features
--------
- Larder Fridge
  - 3 keys : Temp.Setting, Super, ON-OFF
  - Two digits seven-segment
  - Alarm LED
  - Super LED
  - Super mode
  - Ambiance sensor fail mode
  - Evaporator sensor fail mode
  - Ambiance + Evaporator sensors fail mode
  - NTC 2k7 sensor
  - Setpoints 1~8 C
  - Auto test mode
  - State mode
  - Compressor, Fan, Lamp control
- 4-Stars Fridge
  - 3 keys : Temp.Setting, Super, ON-OFF
  - Two digits seven-segment
  - Alarm LED
  - Super LED
  - Super mode
  - Ambiance sensor fail mode
  - Evaporator sensor fail mode
  - Ambiance + Evaporator sensors fail mode
  - NTC 2k7 sensor
  - Setpoints 1~8 C
  - Auto test mode
  - State mode
  - Compressor, Lamp control
- No-Frost Freezer
  - 3 keys : Temp.Setting, Super, ON-OFF
  - Two digits seven-segment
  - Alarm LED
  - Super LED
  - Super mode
  - Defrost mode
  - Ambiance sensor fail mode
  - NTC 2k7 sensor
  - Setpoints -16~-24 C
  - Auto test mode
  - State mode
  - Compressor, Heater, Lamp control
